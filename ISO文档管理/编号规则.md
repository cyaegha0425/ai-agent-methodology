# ISO文档编号体系

## 完整文件名格式

```
{类型}[{部门}]-{编号3位}-{日期YYYYMMDD}-{当天流水号3位}-{文档名称} {版本号}.{扩展名}
```

**示例：**
```
HF-UM-001-20260511-001-铁丝双头阀-使用说明书 V1.0.docx
HF-SOP-MFG-001-20260512-001-产品组装SOP V1.0.docx
HF-FRM-QA-003-20260512-001-铁丝双头阀-出厂检测报告 V1.0.docx
```

---

## 编号规则

| 层级 | 文件类型 | 编号前缀 | 示例 |
|------|----------|----------|------|
| 1阶 | 手册/方针 | HF-MGMT-xxx | HF-MGMT-001 质量手册 |
| 2阶 | 程序文件 | HF-PROC-xxx | HF-PROC-001 文件控制程序 |
| 3阶 | 作业指导书/SOP | HF-SOP-xxx | HF-SOP-MFG-001 出厂检验SOP |
| 4阶 | 表单/记录 | HF-FRM-xxx | HF-FRM-001 检验记录表 |
| 技术文件 | 设计图/BOM等 | HF-TECH-xxx | HF-TECH-001 硬件原理图 |
| 用户手册 | 说明书 | HF-UM-xxx | HF-UM-001 智慧阀门说明书 |

> xxx = 3位序号，如001、002...

## 说明
- HF = 公司标识前缀
- 各系列序号独立从001开始
- 编号变更/作废需记录到台账备注栏
- 日期格式：YYYYMMDD
- 流水号：同编号+同日期共用，从001递增
