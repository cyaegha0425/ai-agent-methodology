# AI协作驱动开发方法论

AI协作驱动开发方法论（Methodology for AI-Cooperative Driven Development）是一套基于Coze平台的多Agent协作开发框架。

## 核心内容

- **v1.3.md** - 完整版方法论（推荐）
- **入门手册.md** - 快速入门指南
- **百科.md** - 详细参考文档

## 目录结构

```
├── 基础设定/          - Agent基础配置
├── 知识库/            - 项目知识库（RSI星际公民、工具技巧、Agent运维等）
├── 文档工作/          - 文档产出目录
└── v1.2归档/          - v1.2历史版本
```

## 快速开始

1. 阅读 入门手册.md 了解核心概念
2. 查看 百科.md 获取详细参考
3. 查阅 知识库/INDEX.md 定位具体问题
