# 日报工作流

## 一、格式规范

### 1.1 今日工作格式
`项目--子项目: 内容；用时X小时；完成X%`

**示例：**
```
智慧球阀--无三通双头阀: 兰州项目说明书修改；用时2小时；完成100%
施肥机--控制器: 问题梳理排查；用时2小时；完成80%
其他--集团技术支持: 技术支持响应；用时2小时；完成90%
其他--文档模板化及生产助手: 文档结构规划；用时2小时；完成70%
```

### 1.2 明日工作计划格式
一行，逗号分隔，**不带序号、不带项目前缀**

**示例：**
```
无三通双头阀说明书书写，跟进控制器问题排查及技术支持，文档模板化
```

### 1.3 需要协调事项
单独问一句"有需要协调的吗"，有则拼到明日后面，无则不填

---

## 二、内容划分

### 2.1 项目-子项目映射（分类铁律）
用户说关键词时，按以下映射归类：

| 用户说的关键词 | 归到 | 示例 |
|---|---|---|
| 视觉识别、树莓派视觉方案、代码修改、工装、检测设备 | 智慧球阀--通用 | 视觉识别方案落地验证 |
| E3、易耘E3 | 施肥机--易耘E3 | E3样机供应链沟通 |
| E1、易耘E1 | 施肥机--易耘E1 | — |
| 软管泵、集成软管泵 | 施肥机--集成软管泵 | — |
| 双阀头、无三通双头阀 | 智慧球阀--无三通双头阀 | — |
| 物料编码、培训、集团协助、杂项 | 其他 | 物料编码添加 |

### 2.2 完整项目列表
- 智慧运营中心项目
- 蓄收能力提升项目
- 智慧球阀（子项目：视觉识别/通用/双阀头/无三通双头阀等）
- 施肥机（子项目：集成软管泵/易耘E1/易耘E3等）
- 变频柜
- 其他--集团技术支持
- 其他--文档模板化及生产助手

---

## 三、工时规则
- 工时合计**必须凑满8小时**
- 工时不够8小时时，**主动问用户补哪项**
- 不要自己编项目凑工时

---

## 四、流程
1. 日程提醒 → 问用户今天干了啥
2. 用户回复 → 按格式+映射归类整理
3. 输出两个代码块供用户确认
4. 用户修改 → 改完立即重新输出完整代码块
5. 确认无误 → 存入 `./日报/YYYY-MM-DD.md`

---

## 五、注意事项
- 今日工作**不需要序号**，直接按行输出
- 内容简洁，一句话概括即可
- 项目数量按实际，用户说几个就写几个
