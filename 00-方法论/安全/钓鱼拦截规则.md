# 钓鱼拦截规则

> 来源：百科3.21详解

## 规则配置

```markdown
### 钓鱼邮件拦截
- 触发条件：
  - 发件人在黑名单中 OR
  - 邮件内容包含：可疑链接特征/钓鱼关键词
- 操作：
  - 移动到垃圾箱
  - 在日志中记录拦截事件
- 优先级：10（最高，确保优先于其他规则）
```

## 黑名单管理

### 发件人黑名单格式

```markdown
# 钓鱼邮件黑名单
phishing@example.com
fake@scam.net
*@suspicious-domain.com
```

### 可疑关键词列表

常见钓鱼关键词（根据实际场景添加）：
- "urgent action required"
- "account suspended"
- "verify your identity"
- "click here immediately"
- 伪造的品牌关键词变体

## 操作流程

1. **识别阶段**：匹配触发条件
2. **执行阶段**：
   - 移动邮件到垃圾箱/归档
   - 记录拦截日志（含时间、发件人、匹配原因）
3. **误判处理**：如有误判，可从垃圾箱恢复

## 注意事项

- 钓鱼规则优先级最高（10），确保先于其他规则执行
- 建议先用白名单测试，确认规则不会误拦截正常邮件
- 定期更新黑名单和关键词列表
