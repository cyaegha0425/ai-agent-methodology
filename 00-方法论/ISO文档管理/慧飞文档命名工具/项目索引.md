## 技术说明

### LocalStorage容量
- 限制：约5MB（浏览器标准）
- 估算：编号库100条约20KB，文档库1000条约500KB，绰绰有余

### 文件名格式
```
{类型}[{部门}]-{编号}-{日期}-{流水号}-{文档名称}
```
- HF-UM类型无部门：`HF-UM-001-20260512-001-智慧阀门-使用说明书`
- 其他类型有部门：`HF-SOP-MFG-001-20260512-001-产品组装SOP`

### 编号库数据结构
```javascript
{
  id: '001',           // 3位编号
  name: '文件控制程序', // 文档名称
  type: 'HF-PROC',    // 类型
  dept: '',            // 部门后缀（无部门则空）
  isoClause: '7.5.3'  // ISO条款（仅HF-PROC有）
}
```
